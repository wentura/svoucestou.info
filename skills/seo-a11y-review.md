# SEO + A11Y review

## SEO kontrola
- Host v `metadataBase`, `openGraph.url`, `robots`, `sitemap` je jednotný.
- Sitemap obsahuje pouze veřejné routy.
- `canonical` je nastavena na veřejných stránkách.
- Metadata nevystavují placeholder stránky.

## A11Y kontrola
- Formuláře mají labely.
- Chybové stavy mají čitelné hlášky.
- Interaktivní prvky mají srozumitelný text.
- Hlavní stránky mají jedno `h1`.
